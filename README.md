FastAPI Project

My this project FastAPI-based backend application with structures layout of API handling,core logic, and UI integration.

It is designed to demonstrate how to build and organize a  scalable backend using FastAPI

TECH:-
Python
FastAPI
Uvicorn
Streamlit

Setup & Installation
1. Clone the repository

git clone https://github.com/Ctrl-Alt-Eliteee/practice-2.git

2. Navigate into project

cd FASTAPI

3. Create virtual environment

python -m venv venv

4. Activate environment

Windows:
venv\Scripts\activate

5. Install dependencies

pip install -r requirements.txt

Running the App

uvicorn api.main --reload

Then open:
http://127.0.0.1:8000
